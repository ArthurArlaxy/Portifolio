Flappy Bird Personalizado

Um clone do famoso jogo Flappy Bird, desenvolvido em Python utilizando a biblioteca Pygame e conceitos de Programação Orientada a Objetos (POO). Este projeto apresenta um personagem diferente em vez do pássaro clássico.

Número Secreto

Neste jogo, você precisa adivinhar um número secreto entre 1 e 100. A cada tentativa, você receberá uma dica se o número secreto for maior ou menor do que a sua tentativa. O objetivo é adivinhar o número com o menor número de tentativas possíveis.
